# Otimização de Custos na Abstergo Industries com AWS

## 📋 Introdução
A Abstergo Industries, empresa farmacêutica líder em inovação, busca reduzir custos operacionais. Este projeto implementa três serviços da AWS para otimizar armazenamento, processamento e instâncias de computação, tornando a nuvem mais acessível e eficiente.

## ☁️ Serviços Implementados

### Amazon S3
**Foco:** Armazenamento de dados seguros e escalável.  
**Caso de uso:** Armazenar backups de dados farmacêuticos e relatórios de pesquisa, reduzindo custos de servidores locais.

### AWS Lambda
**Foco:** Computação sem servidor para executar código sob demanda.  
**Caso de uso:** Processar análises de dados de ensaios clínicos automaticamente, pagando apenas pelo tempo de execução.

### Amazon EC2 Savings Plans
**Foco:** Descontos em instâncias EC2 para uso consistente.  
**Caso de uso:** Otimizar custos de servidores virtuais usados em simulações de medicamentos, garantindo economia previsível.

## 💰 Benefícios Esperados
- Redução de até 30% em custos de infraestrutura.  
- Escalabilidade automática sem intervenção manual.  
- Maior foco em inovação farmacêutica, não em manutenção de TI.
